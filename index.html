assemble-bootstrap

A template for building a static site using bower, grunt, and various grunt tasks, including assemble.

View on github


Features

  • Build tasks managed using Grunt
  • Preview server with LiveReload using grunt-contrib-connect.
  • HTML generated from handelbars templates using Assemble.
  • JS and CSS concatenation and minifications using grunt-usemin
  • Front-end components managed using Bower.
  • Base template using Bootstrap 3.0
